# 高中学习助手 v2.0

游戏化知识点诊断与强化训练系统，支持物理和化学双学科，专为高中学生设计。

## 快速开始

双击 `index.html` 即可在浏览器中打开，无需安装任何软件。

## 学科支持

| 学科 | 知识点数 | 覆盖内容 |
|------|---------|---------|
| 物理 | 17 | 初中前置(5) + 必修一(12)：运动描述、匀变速、相互作用、牛顿定律 |
| 化学 | 15 | 必修一(10)：物质分类、离子/氧化还原、钠氯、物质的量、铁、原子结构/周期律/化学键 + 必修二(5)：硫氮、反应能量、有机物 |
| *数学* | *可扩展* | *预留接口，创建 js/subjects/math.js 即可添加* |
| *生物* | *可扩展* | *预留接口，创建 js/subjects/biology.js 即可添加* |

## 功能

- **统一学科入口**：首次启动显示学科选择页，学习过程中可随时切换学科
- **前置溯源诊断**：从高中知识点向下追溯到前置断链点，定位真正的薄弱环节
- **知识星图**：知识点可视化依赖图，点击节点即可开始诊断
- **计时器+速度奖励**：每道题计时，速度进步有额外 XP 奖励
- **游戏化系统**：XP/等级/徽章/连续天数/金币双重奖励
- **Boss 章节挑战**：掌握知识点后解锁章节综合测试
- **综合考试**：限时全科考试模式
- **奖励商店**：学生用金币兑换奖励，家长审批
- **家长中心**：PIN 保护，管理奖励物品，查看学习数据
- **学科间数据隔离**：每个学科独立的进度、金币、徽章

## 技术架构

```
├── index.html              # 主页面（双击运行，含 Neumorphism CSS）
├── js/
│   ├── data.js             # 物理知识图谱+题库（启动时加载）
│   ├── imported_questions.js  # 外部导入物理真题
│   ├── boost_questions.js  # 物理补充中等难度题
│   ├── items.js            # 道具系统
│   ├── storage.js          # localStorage 读写（按学科隔离）
│   ├── gamification.js     # XP/等级/徽章/连续天数（多学科称号）
│   ├── rewards.js          # 奖励商店+家长中心
│   ├── diagnosis.js        # 前置溯源诊断引擎（学科无关）
│   ├── practice.js         # 练习系统（学科无关）
│   ├── boss.js             # Boss 挑战（各学科独立配置）
│   ├── exam.js             # 综合考试（学科无关）
│   ├── ui.js               # UI 渲染（学科感知）
│   ├── app.js              # 主入口+路由
│   └── subjects/
│       ├── loader.js       # 学科加载器+切换逻辑
│       └── chemistry.js    # 化学知识图谱+题库（15节点）
└── README.md
```

## 扩展现有学科

1. 在 `js/subjects/` 下创建新学科数据文件（参考 `chemistry.js`）
2. 在 `js/subjects/loader.js` 的 `registry` 中添加学科配置
3. 在 `js/gamification.js` 添加学科特定称号和徽章池
4. 在 `js/boss.js` 添加学科特定 Boss 配置
5. 在 `index.html` 学科选择页添加学科卡片
