# 高一物理学习小助手

游戏化物理知识点诊断与强化训练系统，专为厌学学生设计。

## 快速开始

双击 `index.html` 即可在浏览器中打开，无需安装任何软件。

## 功能

- **前置溯源诊断**：从高中知识点向下追溯到初中断链点，定位真正的薄弱环节
- **知识星图**：17个知识点（5个初中前置+12个高中必修一）的可视化依赖图
- **计时器+速度奖励**：每道题计时，速度进步有额外XP奖励
- **游戏化系统**：XP/等级/15种徽章/连续天数/金币双重奖励
- **奖励商店**：学生用金币兑换奖励，家长审批
- **家长中心**：PIN保护，管理奖励物品，查看学习数据

## 文件结构

```
├── index.html          # 主页面（双击运行）
├── js/
│   ├── data.js         # 知识图谱+题库（17节点，~150题）
│   ├── storage.js      # localStorage读写
│   ├── gamification.js # XP/等级/徽章/连续天数
│   ├── rewards.js      # 奖励商店+家长中心
│   ├── diagnosis.js    # 前置溯源诊断引擎
│   ├── practice.js     # 练习系统
│   ├── ui.js           # UI渲染（8屏）
│   └── app.js          # 主入口
└── README.md
```

## 题库说明

当前题库基于中国高中物理课程标准生成。后期可从以下来源替换为真实真题：
- 第一试卷网 (shijuan1.com) - 免费下载Word试卷
- 国家中小学智慧教育平台 (smartedu.cn) - 约1.5万道免费题
- 人人文库 (renrendoc.com) - 136页完整同步练习PDF
- phy-10k 数据集 (ieee-dataport.org) - 数万道标注知识点的真题

将下载的题目文件放入 `data/` 目录后可编写导入脚本批量替换。
